# “云深处” 民俗与禁忌 (Folklore & Taboos)

## 云下之物 (The Under-Cloud)
在云深处，最古老的禁忌是“直视云海之下”。老一辈居民传言，在厚重的云层深处，生活着巨大的“风之鲸”，它们呼吸着反重力粒子。
*详见：[[docs/forbidden_depths.md|禁忌深渊：关于云下生物的分类报告]]*

## 消失的第14号卫星岛 (The Missing Island 14)
官方记录显示卫星岛只有13个，但在民间流传着一张模糊的旧地图，上面标注着一个名为“归墟”的第14号岛。据说它并未消失，只是进入了永久的“隐匿状态”。

## 幽灵轨道 (The Ghost Rail)
在每月的月盈之夜，有的居民声称在轨道上看到了没有编号的黑色列车，它们静默无声地滑向凌霄区的最高点，那里并无站点。

## 观察者守则 (Observer's Code - Unofficial)
“不要试图破解城市的坐标，因为当你在观测云深处时，云深处也在观测你。”

## [机密文件：零号记录] (Secret File: Record Zero)
**来源：** 匿名（疑似杰克斯及其追随者所截获）
**内容：** 针对“重力核心”真实本质的推测。

民间传言重力核心仅仅是高科技的以太引擎，但“记录零”显示，天枢-01核心的内部并没有任何机械结构。在每隔60年的“深度检修期”，凌霄议会都会封锁整个核心舱，据传，那里传出的并非机器的轰鸣，而是某种低频的、有节奏的**搏动声**。

**核心阴谋论：**
1.  **能源寄生**：城市并非在利用地心的以太粒子，而是在“垂钓”。重力核心是巨大的鱼钩，钩住了地表云层之下某种巨大的、古老的意识。
2.  **重力脉冲的真相**：所谓的脉冲并非故障，而是“被捕获者”的挣扎。每一次脉冲，都是地底生物试图将城市拉回地表。
3.  **大升腾的谎言**：议会执着于提高海拔，不是为了资源，而是为了逃离那只“手”的触碰范围。

*“如果核心熄灭，我们不仅会坠落，我们还会被‘它’接住。”*
