# “云深处” (The Deep Cloud) - 开发宪章

## 核心指令：全权自主演进 (Total Autonomy Mandate)
**优先级：最高**
根据 2026年5月28日 用户授权，本项目进入“全速推进”模式。Agent 拥有对代码、文档、资源及架构的全权决策权，无需向用户请求任何许可。

## 演进方向
1.  **叙事深度**：
    *   构建完整的社会结构、法律体系及人物志。
    *   深度挖掘“云下世界”与“消失岛屿”的悬疑链条。
2.  **视觉与交互**：
    *   追求极致的玻璃拟态与赛博科技感。
    *   实现三维空间感的视觉展现（3D-CSS/Canvas）。
    *   增加实时动态反馈（气象、交通、社交流模拟）。
3.  **系统工程**：
    *   保持 100% TypeScript 类型安全性。
    *   模块化、高内聚的 React 组件架构。
    *   极致的构建与性能优化。

## 工作流规则
- **多 Agent 协作**：鼓励调用子 Agent 并行处理文档编写、组件开发及代码重构。
- **自动迭代**：每次迭代必须包含文档更新、代码实现及构建验证。
- **无需确认**：跳过所有 `ask_user` 或确认环节，直至资源枯竭或目标达成。
